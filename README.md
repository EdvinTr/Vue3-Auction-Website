# Vue-Auction-Website

### Technologies used
**Frontend:**
* Vue 3
* vue-router
* vuex
* vuex-persistedstate
* vue3-google-oauth2
* Tailwind CSS
* axios

**Backend:**
* Java Spring
* Spring Data JPA
* Spring Security
* Spring Web
* Spring MySQL Connector
* Spring mail
* Spring validation
* Lombok
